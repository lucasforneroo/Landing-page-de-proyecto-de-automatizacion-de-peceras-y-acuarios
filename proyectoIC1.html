Pecera Inteligente con ESP32

Automatiza y controla tu pecera desde cualquier lugar. Monitorea temperatura, controla iluminación y cuida a tus peces con tecnología IoT de última generación.

Quiero saber más

¿Qué es este proyecto?

Diseñamos un sistema automatizado para peceras, controlado por un ESP32, que permite:

  • Medir la temperatura del agua en tiempo real.
  • Encender/apagar bomba de aire o resistencia calefactora automáticamente.
  • Regular la iluminación LED simulando ciclos de día y noche.
  • Controlar todo desde una app móvil con datos en vivo y control manual.

¡Haz tu pecera más segura, autónoma y eficiente!

¿Cómo lo vamos a hacer?

Sensado

  • Sensor DS18B20 mide la temperatura del agua.
  • ESP32 lee y procesa los datos.

Actuaciones

  • Relé digital para bomba/resistencia.
  • PWM para regular brillo de LEDs.

Conectividad y App

  • ESP32 se conecta por WiFi.
  • Datos y control desde app móvil (Blynk o web propia).

Integración

  • Todo programado en Arduino IDE o Micropython.
  • Prototipo funcional para exposición.

¿Con qué lo vamos a hacer?

  • ESP32 (microcontrolador principal)
  • Sensor DS18B20 sumergible
  • Módulo relé (bomba/resistencia)
  • Tira o lámpara LED regulable
  • Fuente de alimentación 5V/12V
  • App móvil (Blynk o propia)

¿Cómo funciona?

  1. El ESP32 se conecta al WiFi y lee la temperatura del agua.
  2. Si la temperatura baja de 24°C, activa la resistencia o bomba.
  3. Si sube de 26°C, apaga la resistencia o bomba.
  4. La iluminación LED se regula automáticamente o desde la app.
  5. Desde la app puedes ver datos en tiempo real y controlar todo manualmente.

Resumen:

  • Controlador inteligente para peceras con ESP32.
  • Automatización y control remoto vía WiFi.
  • Mejor cuidado y seguridad para tus peces.
